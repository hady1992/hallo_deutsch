# Supabase RLS Notes

هذه المنصة تستخدم Supabase لتخزين وإدارة محتوى قابل للتعديل من لوحة الإدارة.

## الجداول المستخدمة

- `exercises`
- `vocabulary`
- `exams`
- `placement_tests`

## ملاحظة أمنية

حماية الواجهة الأمامية وحدها غير كافية. حتى لو كانت صفحة `/admin` محمية بتسجيل دخول وفلترة بريد المدير، يمكن لأي مستخدم متقدم محاولة التعامل مع Supabase مباشرة من المتصفح إذا لم تكن سياسات قاعدة البيانات مفعلة.

يجب تفعيل Row Level Security (RLS) على الجداول أعلاه قبل الاعتماد على التخزين السحابي في الإنتاج، مع سياسات تمنع الكتابة والحذف إلا للمستخدمين الإداريين المصرح لهم.

لا يحتوي هذا الملف على SQL، ولا يغيّر قاعدة البيانات. الهدف منه توثيق خطوة النشر المطلوبة فقط.
