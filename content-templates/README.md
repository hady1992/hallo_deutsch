# قوالب استيراد المحتوى — Hallo Deutsch

هذا المجلد يحتوي قوالب JSON رسمية جاهزة للاستخدام مع نظام استيراد المحتوى بلوحة التحكم (`/admin`). كل قالب يحتوي 2-3 أمثلة صحيحة وقابلة للاستيراد مباشرة، بالحقول **مطابقة تمامًا** لما يقبله نظام الاستيراد الحالي بالكود.

> ✅ **تحديث**: جميع القوالب الثمانية بهذا المجلد صارت الآن قابلة للاستيراد فعليًا كملف JSON مباشرة عبر أزرار رفع حقيقية بلوحة التحكم أو الصفحات المعنية (بما فيها التمارين واختبار تحديد المستوى، اللذان كانا يدعمان CSV فقط أو الإدخال اليدوي فقط سابقًا).

---

## 1) `vocabulary-template.json` — المفردات العامة

**تُستورد من:** `/admin` → قسم "استيراد البيانات الأساسية (محلي)" → عمود "المفردات العامة" (زر رفع ملف، يقبل `.json` و`.csv`).

**تظهر بعد الاستيراد في:** صفحة `/vocabulary` (تبويب "المفردات").

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `german` | الكلمة بالألمانية |
| `translation` | الترجمة العربية (أو استخدم `arabic` بدلًا منه) |
| `type` | نوع الكلمة (مثل `noun`, `verb`, `adjective`) |
| `example` | جملة مثال بالألمانية |
| `exampleArabic` | ترجمة جملة المثال |

### الحقول الاختيارية
`level` (افتراضي `A1`)، `category` (افتراضي "عام")، `pronunciation`، `id`.

### مثال سريع
```json
{
  "german": "das Buch",
  "translation": "الكتاب",
  "type": "noun",
  "level": "A1",
  "example": "Ich lese ein Buch.",
  "exampleArabic": "أنا أقرأ كتابًا."
}
```

## 2) `verbs-template.json` — الأفعال

**تُستورد من:** `/admin` → قسم "استيراد البيانات الأساسية (محلي)" → عمود "الأفعال".

**تظهر بعد الاستيراد في:** تبويب "قاموس الأفعال" بصفحة `/grammar`، وتبويب الأفعال بصفحة `/vocabulary`.

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `german` | مصدر الفعل بالألمانية (أو استخدم `infinitive`) |
| `translation` | الترجمة العربية |
| `type` | نوع الفعل (`weak`/`strong`/غيرها) |

### الحقول الاختيارية
- `conjugations`: كائن (object) بمفاتيح أزمنة مثل `Präsens`, `Perfekt`, وكل زمن يحتوي تصريفات الضمائر (`ich`, `du`, `er`, `sie`, `es`, `wir`, `ihr`, `Sie`). **إن تُرك فارغًا، يُقبل الفعل لكن بدون جدول تصريف.**
- `examples`: كائن بمفاتيح أزمنة، كل زمن يحتوي `{ "de": "...", "ar": "..." }`.
- `id`.

### مثال سريع
```json
{
  "german": "gehen",
  "translation": "يذهب",
  "type": "strong",
  "conjugations": {
    "Präsens": { "ich": "gehe", "du": "gehst", "er": "geht", "sie": "geht", "es": "geht", "wir": "gehen", "ihr": "geht", "Sie": "gehen" }
  }
}
```

## 3) `nouns-template.json` — الأسماء

**تُستورد من:** `/admin` → قسم "استيراد البيانات الأساسية (محلي)" → عمود "الأسماء".

**تظهر بعد الاستيراد في:** تبويب "قاموس الأسماء" بصفحة `/grammar`.

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `german` | الاسم كاملًا مع أداة التعريف (مثل `das Haus`) |
| `translation` | الترجمة العربية |
| `noun` | الاسم بدون أداة التعريف (مثل `Haus`) |
| `gender` | الجنس النحوي (`masculine`/`feminine`/`neuter`) |
| `article` | أداة التعريف، ويجب أن تكون **بالضبط** واحدة من: `der`, `die`, `das` |

### الحقول الاختيارية
`plural` (صيغة الجمع، افتراضي `-`)، `example`، `exampleArabic`، `id`.

### مثال سريع
```json
{
  "german": "das Haus",
  "translation": "المنزل",
  "noun": "Haus",
  "gender": "neuter",
  "article": "das",
  "plural": "die Häuser"
}
```

## 4) `exercises-template.json` — التمارين

**تُستورد من:** `/admin` → قسم "استيراد التمارين" → زر "رفع تمارين (CSV أو JSON)".

**تظهر بعد الاستيراد في:** صفحة `/exercises`، ضمن مستوى `level` والفئة (`category`) المحددين بكل تمرين — الطالب يختار المستوى ثم الفئة وعدد الأسئلة قبل بدء التمرين (لا تظهر كل الأسئلة دفعة واحدة).

✅ **مُحدَّث:** يدعم هذا القسم الآن رفع ملفات **JSON و CSV معًا** (كان CSV فقط سابقًا)، مع حقلين اختياريين جديدين لدعم الترجمة العربية والنطق الصوتي. زر "تحميل قالب JSON" متاح بنفس القسم لتنزيل نسخة جاهزة مطابقة لهذا الملف.

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `question` | نص السؤال بالألمانية (تستخدم الفراغ `___` أو `____` لمكان الإجابة) |
| `options` | قائمة الخيارات (array) |
| `correctAnswer` | الإجابة الصحيحة — نصًا مطابقًا لأحد عناصر `options`، **أو رقم موضع الخيار مباشرة** (كلاهما مقبول، يُحوَّل النص تلقائيًا لرقم الموضع عند الحفظ) |

### الحقول الاختيارية
| الحقل | الوصف |
|---|---|
| `questionArabic` | ترجمة عربية للسؤال، تُعرض تحته بلون أهدأ. إن لم توجد، يُستخدم `translation` ثم `arabic` كبدائل، وإلا لا تظهر أي ترجمة |
| `audioText` | نص ألماني بديل يُقرأ صوتيًا بدل السؤال نفسه (مفيد لو أردت جملة كاملة أوضح نطقًا بدل نص فيه فراغ) |
| `category` | فئة/موضوع القاعدة (مثل `artikel`, `verben`, `sein`, `negation`...) تُستخدم لتقسيم التمارين بصفحة `/exercises`. إن لم توجد، يُستخدم `topic` كبديل، وإلا تُصنَّف تحت "عام" |
| `difficulty` (افتراضي `medium`)، `level` (افتراضي `A1`)، `type` (افتراضي `multipleChoice`)، `explanation`. |

### مثال سريع
```json
{
  "question": "Ich ___ Ahmad.",
  "questionArabic": "اختر الفعل الصحيح.",
  "audioText": "Ich heiße Ahmad.",
  "options": ["bin", "bist", "ist", "sind"],
  "correctAnswer": "bin",
  "level": "A1",
  "category": "sein",
  "explanation": "مع ich نستخدم bin."
}
```

## 5) `exams-template.json` — الامتحانات

**تُستورد من:** `/admin` → قسم "استيراد التمارين"... لا، بل داخل صفحة `/exams` نفسها (زر "رفع امتحان" بأعلى كل تبويب مستوى، مرئي فقط لحساب أدمن). **يدعم JSON و CSV معًا.**

**تظهر بعد الاستيراد في:** صفحة `/exams`، ضمن تبويب المستوى (`level`) المحدد بكل امتحان.

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `title` | عنوان الامتحان |
| `level` | المستوى (`A1`/`A2`/`B1`/`B2`) |
| `questions` | قائمة أسئلة (array)، غير فارغة |
| `questions[].question` | نص السؤال |
| `questions[].options` | قائمة الخيارات (array) |
| `questions[].correctAnswer` | الإجابة الصحيحة، نصًا مطابقًا لأحد عناصر `options` |

### الحقول الاختيارية
`id` (يُنشأ تلقائيًا إن تُرك فارغًا)، `description`، `duration` (بالدقائق، افتراضي 30)، `difficulty`، `questions[].explanation`.

**ملاحظة:** يمكن للملف الواحد أن يحتوي أكثر من امتحان بمستويات مختلفة دفعة واحدة (مصفوفة من الامتحانات)؛ كل امتحان يُحفظ تلقائيًا ضمن تبويب مستواه الصحيح.

### مثال سريع
```json
[
  {
    "id": "exam-a1-001",
    "level": "A1",
    "title": "اختبار A1 تجريبي",
    "questions": [
      {
        "question": "Ich ___ Ahmad.",
        "options": ["bin", "bist", "ist", "sind"],
        "correctAnswer": "bin",
        "explanation": "مع ich نستخدم bin."
      }
    ]
  }
]
```

## 6) `placement-test-template.json` — اختبار تحديد المستوى

**تُستورد من:** `/admin` → قسم "استيراد اختبار تحديد المستوى" → قسم "رفع دفعة أسئلة (JSON)" (أعلى نموذج الإضافة اليدوية).

**تظهر بعد الاستيراد في:** صفحة `/placement-test`.

✅ **مُحدَّث:** أصبح بالإمكان الآن رفع ملف JSON يحتوي عدة أسئلة **دفعة واحدة**، بجانب (وليس بدلًا من) نموذج إضافة سؤال واحد يدويًا الذي كان متاحًا سابقًا. زر "تحميل قالب JSON" متاح بنفس القسم.

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `question` | نص السؤال |
| `options` | قائمة من الخيارات (array) |
| `correctAnswer` | الإجابة الصحيحة — **رقم موضع** ضمن `options` (0 = الأول...)، **أو نص مطابق لأحد الخيارات** (يُحوَّل تلقائيًا لرقم الموضع عند الحفظ) |

### الحقول الاختيارية
`level` (أو `targetLevel` كبديل، افتراضي `A1`)، `explanation`، `id`.

### مثال سريع
```json
{
  "question": "Wie ___ du?",
  "options": ["heißt", "heißen", "hast", "heiße"],
  "correctAnswer": 0,
  "level": "A1"
}
```

## 7) `grammar-template.json` — قواعد اللغة

**تُستورد من:** `/admin` → قسم "قواعد اللغة" → "استيراد القواعد (JSON أو CSV)".

**تظهر بعد الاستيراد في:** تبويب **"قواعد مستوردة"** بصفحة `/vocabulary` — **وليس** بصفحة `/grammar` أو `/grammar/rules` الرئيسيتين، لأن تينك الصفحتين تعرضان محتوى القواعد الثابت المُعَدّ مسبقًا فقط. هذا فرق مهم ومقصود، موضّح أيضًا داخل لوحة التحكم نفسها.

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `title` | عنوان القاعدة |
| `level` | المستوى (`A1`/`A2`/`B1`/`B2`) |
| `explanation` | شرح القاعدة (نص واحد) |

### الحقول الاختيارية
`examples` (قائمة جمل أمثلة، array)، `notes` (ملاحظات إضافية)، `id`.

### مثال سريع
```json
{
  "title": "أدوات التعريف (Der, Die, Das)",
  "level": "A1",
  "explanation": "تُستخدم أدوات التعريف حسب جنس الاسم: der للمذكر، die للمؤنث، das للمحايد.",
  "examples": ["Der Mann ist groß.", "Die Frau ist nett."]
}
```

## 8) `kids-vocabulary-template.json` — مفردات الأطفال

**تُستورد من:** `/admin` ← لا، بل من صفحة `/kids` نفسها → تبويب "استيراد" (مرئي فقط لحساب أدمن) → قسم "تحميل ملفات CSV" (يقبل فعليًا `.json` أيضًا رغم التسمية) → زر "استيراد" بجانب أي فئة (حيوانات، طعام، مواصلات...).

**تظهر بعد الاستيراد في:** صفحة `/kids`، تبويب "كلمات".

### الحقول المطلوبة
| الحقل | الوصف |
|---|---|
| `german` | الكلمة بالألمانية (أو استخدم `word` بدلًا منه) |
| `arabic` | الترجمة العربية (أو استخدم `translation` بدلًا منه) |

### الحقول الاختيارية
- `image`: إيموجي يمثّل الكلمة (افتراضي 📝 إن تُرك فارغًا).
- `category`: **ملاحظة خاصة** — الفئة الفعلية المُسجَّلة للكلمة تُحدَّد فعليًا حسب **أي زر فئة** بالواجهة تضغط عليه أثناء الرفع (حيوانات/طعام/مواصلات...)، وليس بالضرورة من قيمة هذا الحقل بالملف. أدرجناه بالقالب للتوثيق والتنظيم فقط.

### مثال سريع
```json
{
  "german": "Der Hund",
  "arabic": "الكلب",
  "category": "animals",
  "image": "🐶"
}
```
